como programar html

para programar html necesitas un editor de codigo el cual puede ser Visual Estudio Code (clic en lo rojo para descargarlo) este se utiliza tanto en la version de window 10 como en mac y linux. Despues de descargarlo e instalarlo ejecutelo. Habiendo echo esto instalaremos unas librerias las cuales son

Cambio de idiomas:

Spanish Lenguage Pack for Visua

Resfrescado de pagina automatico:

Live Server

Despues de instalar estas extenciones cierre y vuelva a abrir el editor de codigo con esto pulsaremo F1 y le saldra la barra de comandos. En ella escribiremos (Configure Display Lenguage) y seleccionaremos (es). Luego de esto cerraremos y abriremos el editor de codigo. ¡Ya tendremos el Visual Estudio Code en español y configurado!.

Etiquetas

<!DOCTYPE html>: Esta es necesaria para empesar todos los codigos html para indicar al navegador que estemos usando, que tipo de vercion de html estemos usando

<html>: Va despues de la linea de DOCTYPE y determina como se vera un sitio web

¡Los codigos anteriores son necesarios para iniciar un archivo html!

<title>: Sirve para ponerle titulo a la pagina

<head>: Esta sirve para introducir el encabezado de la pagina

<body>: Esto sirve para introducir el cuerpo de la pagina

<p>: esto sirve para introducir un parrafo que ira de lado a lado de la pestaña

<video src="(aqui la direccion del video que deseo ejemplo:si lo tiene en la misma carpeta seria video src="nombre del video.mp4" si lo tiene en otra carpeta dentro de la misma carpera seria video src="nombre de la carpeta/nombre del video.mp4")">: para incluir un video

<img src="nombre de la imagen como se esplica en video " alt="descripcion de la imagen por si no esta disponible">: para incluir una imagen

<style>: Para incluir un archivo css sin necesidad de ser externo

<script>: Para incluir un archivo javascript sin necesidad de ser externo

Esto es un buen comienzo para la programacion con el lenguage HTML
